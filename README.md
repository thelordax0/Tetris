# Tetris

Modern, full HD Tetris oyunu implementasyonu.

## Özellikler
- Full HD (1920x1080) çözünürlük
- Modern arayüz tasarımı
- Yüksek skor sistemi
- FPS göstergesi
- Gradient efektler

## Kontroller
- ← → : Hareket
- ↑ : Döndürme
- ↓ : Hızlı düşürme
- SPACE : Anında düşürme
- P : Duraklat
- ESC : Menüye dön
